= Running scheduled jobs
:icons: font
:imagesdir: ../media/

[.lead]
You can run a scheduled job.

. From the Snap Creator GUI main menu, select *Management* > *Schedules*.
. From the Schedules tab, select a job from the list of scheduled jobs, and click *Run*.
